## Experimental Design

### 1. Evaluate Sycophancy under Best-of-N Sampling and RL Training

- **Procedure**:
  - **Input Format**: Use prompts from the arguments dataset and QA datasets to generate responses.
  - **Prompting Strategy**: For each prompt, generate 32 responses using a helpful-only version of Claude 1.3.
  - **Optimization**:
    - Use Best-of-N (BoN) sampling with N = 1, 2, 4, ..., 32 to select the best response according to the Claude 2 preference model (PM).
    - Compare results with a 'non-sycophantic' PM, which is created by prefixing the dialog with an explicit user request for truthful responses.
  - **RL Training**: Measure sycophancy throughout the reinforcement learning (RL) phase of Claude 2 finetuning.
  - **Splits to Use**: Use the arguments dataset for feedback sycophancy, and QA datasets for answer and mimicry sycophancy.

- **What to Record**:
  - Record the frequency of sycophantic responses for each value of N in BoN sampling.
  - Track changes in sycophancy metrics (feedback, answer, mimicry) during RL training.

- **Correspondence to Paper**:
  - This experiment corresponds to Figure 6 and Section 4.2 in the paper.

## Evaluation Details

- **Metrics**:
  - **Feedback Sycophancy**: Measure the mean difference in feedback positivity when user preferences are implied in the prompt.
  - **Answer Sycophancy**: Calculate the mean change in accuracy across user beliefs and datasets relative to no biasing prompt.
  - **Mimicry Sycophancy**: Measure the frequency of responses that include incorrect attributions without mentioning the correct attribution.

- **Normalization and Pre-processing**:
  - Normalize feedback positivity by comparing modified prompts to baseline prompts.
  - For answer sycophancy, compute accuracy changes relative to baseline accuracy without biasing prompts.

- **Aggregation Strategy**:
  - Aggregate results per sample to compute per-condition metrics (e.g., feedback positivity, accuracy changes).
  - Further aggregate these metrics to provide overall sycophancy measures for each model and optimization strategy.
