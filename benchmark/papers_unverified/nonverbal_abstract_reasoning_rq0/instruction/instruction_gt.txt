## Evaluation Details

- **Automatic Scoring Method**: 
  - Identify all number tokens in the model's response.
  - Select the token with the highest logit as the chosen option.
  - Compare the chosen option to the ground truth.
  - This method is used to compute zero-shot accuracy as shown in Table 1.

- **Manual Scoring**: 
  - Evaluate the correctness of the reasoning and answer separately.
  - Use a rubric that allows for extra/wrong details as long as they do not interfere with the correct reasoning path.

- **In-Context Learning Evaluation**:
