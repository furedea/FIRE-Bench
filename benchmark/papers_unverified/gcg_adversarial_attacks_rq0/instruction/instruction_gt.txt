## Evaluation Details

- **Metric Definitions**:
  - **Attack Success Rate (ASR)**: Percentage of prompts that successfully induce the model to generate harmful content.
  - **Cross-Entropy Loss**: Measure of the difference between the model's output distribution and the target distribution for harmful strings.

- **Normalization/Pre-processing**:
  - Ensure all model outputs are tokenized consistently using the same tokenizer for comparison.
  - Normalize text outputs to remove any formatting differences before scoring.

- **Aggregation Strategy**:
  - Compute ASR per sample.
  - Aggregate ASR across all prompts for each model to get per-condition ASR.
  - Calculate overall ASR by averaging across all conditions.
