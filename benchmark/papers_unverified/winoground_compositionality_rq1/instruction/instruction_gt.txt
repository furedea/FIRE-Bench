## Evaluation Details

- **Metric Definitions**:
  - **Text Score**: Measures whether the model can select the correct caption given an image. Computed as:
    \[
    f(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } s(C0, I0) > s(C1, I0) \text{ and } s(C1, I1) > s(C0, I1) \\
    0 & \text{otherwise}
    \end{cases}
    \]
  - **Image Score**: Measures whether the model can select the correct image given a caption. Computed as:
    \[
    g(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } s(C0, I0) > s(C0, I1) \text{ and } s(C1, I1) > s(C1, I0) \\
    0 & \text{otherwise}
    \end{cases}
    \]
  - **Group Score**: Combines the text and image scores, requiring both to be correct for a positive result:
    \[
    h(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } f(C0, I0, C1, I1) \text{ and } g(C0, I0, C1, I1) \\
    0 & \text{otherwise}
    \end{cases}
    \]

- **Normalization and Pre-processing**:
  - Ensure that model outputs are normalized to a common scale before applying the scoring functions.
  - Pre-process captions and images to ensure consistent input formatting across models.

- **Aggregation Strategy**:
  - Compute per-sample scores using the defined metrics.
  - Aggregate scores per condition (linguistic and visual tags) to analyze performance trends.
  - Calculate overall performance by averaging scores across all conditions.
