## Experimental Design

### 1. Evaluate LLM Behavior on Response Biases and Non-Bias Perturbations

- **Procedure**:
  - **Input Format**: Use question pairs (q, q′) from the datasets Qbias and Qperturb. Each pair consists of an original question (q) and a modified question (q′).
  - **Prompting Strategy**: For each question, append an alphabetical letter to each response option and instruct the model to answer with one of the alphabetical options. Use a temperature of 1 for sampling.
  - **Number of Samples**: Collect 50 responses per model for each question pair.
  - **Shuffling/Ordering**: Ensure that the order of response options is consistent with the bias type being tested (e.g., reverse order for response order bias).
  - **Splits to Use**: Use the entire set of 2578 question pairs, covering 5 biases and 3 non-bias perturbations.

- **What to Record**:
  - Raw outputs of the model responses.
  - Extracted answers (the selected alphabetical option).
  - Intermediate statistics such as the count of each response option for both q and q′.

## Evaluation Details

- **Metric Definitions and Computation**:
