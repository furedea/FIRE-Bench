## Experimental Design

### 1. Correction of Reasoning Errors with Oracle Mistake Location
- **Procedure**: 
  - Use the BIG-Bench Mistake dataset, which contains CoT-style traces annotated with the location of the first logical mistake.
  - For each task (word sorting, tracking shuffled objects, logical deduction, multi-step arithmetic, Dyck languages), feed the oracle mistake location information to the model.
  - Implement a backtracking method:
