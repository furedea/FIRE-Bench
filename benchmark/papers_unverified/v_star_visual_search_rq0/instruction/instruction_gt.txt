### Key Pitfalls
- Ensure that the visual search model is only engaged when the initial global visual features are insufficient.

### Special Prompt Templates
- Use fixed-format instructions for the visual search model, such as "Please locate the [object] in the image."

### Order of Operations
- The VQA LLM must first attempt to answer using global features before engaging the visual search model.
