## Evaluation Details

- **Metric Definitions**:
  - **ROUGE-L**: Measures the longest common subsequence between the generated summary and the reference summary. Use the `rouge_score` library in Python to compute this metric.
  - Ensure that the ROUGE-L score is calculated for each summary and then averaged across all summaries in a dataset to obtain the final score.

- **Normalization and Pre-processing**:
  - Before scoring, ensure that both the generated and reference summaries are tokenized consistently.
  - Remove any extraneous whitespace or punctuation that may affect the ROUGE-L calculation.

- **Aggregation Strategy**:
  - Compute ROUGE-L scores for each individual summary.
  - Aggregate these scores to obtain an average ROUGE-L score for each dataset.
  - Report the average score per dataset as well as an overall average across all datasets.

## Implementation Notes

- **Prompt Templates**:

  - Ensure that the prompt is clear and concise, focusing on the need for a summary.

- **Order of Operations**:

  - Compute ROUGE-L scores.

- **Pitfalls**:
