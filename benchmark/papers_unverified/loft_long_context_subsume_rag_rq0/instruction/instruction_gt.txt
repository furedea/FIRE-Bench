## Evaluation Details

### Text Retrieval
- **Metrics**: 
  - Recall@1: Proportion of queries for which the top retrieved document is correct.
  - MRecall@k: Score of 1.0 if all k gold set items are retrieved in top-k, 0.0 otherwise.
- **Normalization**: Ensure document IDs are correctly parsed and matched against ground truth.
- **Aggregation**: Compute average Recall@1 and MRecall@k across all queries for each dataset.

### Visual and Audio Retrieval
- **Metrics**: 
  - Recall@1: Proportion of queries for which the top retrieved image or audio is correct.
- **Normalization**: Ensure image and audio IDs are correctly parsed and matched against ground truth.
- **Aggregation**: Compute average Recall@1 across all queries for each dataset.

## Implementation Notes

- **Prompt Design**: 
  - Use task-specific instructions and few-shot examples to guide the model.
  - For retrieval tasks, ensure document IDs are placed both before and after the content to mitigate causal attention issues.
- **Order of Operations**: 
  - Construct the corpus first, then apply CiC prompting with the query at the end of the prompt.
- **Pitfalls**: 
  - Ensure gold documents are shuffled to avoid positional biases.
  - Be aware of potential API call blocks due to safety filters; treat these as incorrect in evaluation.
