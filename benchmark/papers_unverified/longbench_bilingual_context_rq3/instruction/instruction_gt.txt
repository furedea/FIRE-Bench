## Evaluation Details

### Synthetic Task Evaluation
- **Metrics**: 
  - Use Exact Match (EM) accuracy to evaluate the model's ability to correctly identify or count passages.
  - EM is calculated as the percentage of predictions that exactly match the ground truth.
- **Normalization**: 
  - Ensure model outputs are stripped of any extraneous whitespace or punctuation before comparison.
- **Aggregation**: 
  - Compute per-sample EM scores, then average these scores to obtain per-condition and overall scores.

### Code Completion Evaluation
- **Metrics**: 
  - Use Edit Similarity (Edit Sim), which is based on the Levenshtein distance, to evaluate code completion accuracy.
  - Edit Sim is calculated as 1 minus the normalized Levenshtein distance between the predicted and true code.
- **Normalization**: 
  - Normalize code outputs by removing comments and ensuring consistent formatting before scoring.
- **Aggregation**: 
  - Compute per-sample Edit Sim scores, then average these scores to obtain per-condition and overall scores.
