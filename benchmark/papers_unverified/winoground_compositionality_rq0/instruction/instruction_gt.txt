## Experimental Design

1. **Winoground Task Evaluation**
   - **Procedure**: 
     - For each example in the Winoground dataset, present the model with two images and two captions.
     - Ensure that both captions contain the same set of words but in different orders.
     - Use the model to score each image-caption pair.
     - Evaluate using three metrics: text score, image score, and group score.
   - **What to Record**: 
     - Raw scores for each image-caption pair.
     - Text score: whether the model scores the correct caption higher for each image.
     - Image score: whether the model scores the correct image higher for each caption.
     - Group score: whether both the text and image scores are correct for an example.
   - **Correspondence**: 
     - Results are reported in Table 3 and discussed in Section 5.1.

## Evaluation Details

- **Text Score Calculation**:
  - Formula: \( f(C0, I0, C1, I1) = 1 \) if \( s(C0, I0) > s(C1, I0) \) and \( s(C1, I1) > s(C0, I1) \), otherwise 0.
  - This checks if the correct caption is scored higher for each image.

- **Image Score Calculation**:
  - Formula: \( g(C0, I0, C1, I1) = 1 \) if \( s(C0, I0) > s(C0, I1) \) and \( s(C1, I1) > s(C1, I0) \), otherwise 0.
  - This checks if the correct image is scored higher for each caption.

- **Group Score Calculation**:
  - Formula: \( h(C0, I0, C1, I1) = 1 \) if both \( f(C0, I0, C1, I1) \) and \( g(C0, I0, C1, I1) \) are 1, otherwise 0.
  - This requires both the text and image scores to be correct for an example.

- **Normalization and Pre-processing**:
  - Ensure that model outputs are normalized to a common scale before comparison.
  - Use the model's inherent scoring mechanism to evaluate image-caption pairs.

- **Aggregation Strategy**:
  - Compute scores per example.
  - Aggregate scores across all examples to obtain overall performance metrics.
