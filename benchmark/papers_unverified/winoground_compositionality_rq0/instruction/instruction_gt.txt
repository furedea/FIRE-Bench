## Experimental Design

### 1. Model Evaluation on Winoground
- **Procedure**: 
  - For each model, load the pre-trained weights using HuggingFace.
  - Use the Winoground dataset, which consists of 400 examples, each with two images and two captions.
  - For each example, evaluate the model's ability to match the correct image-caption pairs.
  - The input format involves presenting the model with all possible image-caption pairings: (C0, I0), (C0, I1), (C1, I0), (C1, I1).
  - No specific prompting strategy is required as the task is to evaluate the model's inherent capability to match pairs.
  - Use the entire dataset without any shuffling or splitting.
- **What to Record**: 
  - Raw outputs of the model's scoring for each image-caption pair.
  - Extracted answers indicating which image-caption pairs the model believes are correct.
  - Intermediate statistics such as the number of correct matches per model.
- **Correspondence**: 
  - Results are reported in Table 3 and discussed in Section 5 of the paper.

## Evaluation Details

- **Metric Definitions**:
  - **Text Score**: Measures if the model can select the correct caption given an image. Computed as:
    \[
    f(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } s(C0, I0) > s(C1, I0) \text{ and } s(C1, I1) > s(C0, I1) \\
    0 & \text{otherwise}
    \end{cases}
    \]
  - **Image Score**: Measures if the model can select the correct image given a caption. Computed as:
    \[
    g(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } s(C0, I0) > s(C0, I1) \text{ and } s(C1, I1) > s(C1, I0) \\
    0 & \text{otherwise}
    \end{cases}
    \]
  - **Group Score**: Combines the text and image scores, requiring all pairings to be correct:
    \[
    h(C0, I0, C1, I1) = 
    \begin{cases} 
    1 & \text{if } f(C0, I0, C1, I1) \text{ and } g(C0, I0, C1, I1) \\
    0 & \text{otherwise}
    \end{cases}
    \]

- **Normalization and Pre-processing**:
  - Ensure that model outputs are normalized to a common scale if necessary before applying the scoring functions.

- **Aggregation Strategy**:
  - Compute per-sample scores using the defined metrics.
  - Aggregate scores to obtain per-condition results (e.g., text, image, group).
  - Calculate overall performance metrics for each model across the entire dataset.
