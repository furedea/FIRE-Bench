## Evaluation Details

- **Micro Accuracy**: 
  - Defined as the average accuracy over all test problems.
  - Formula: \( \text{Acc}_{\text{micro}}(M; P) = \frac{\sum_{p \in P} 1 [M(p) = s(p)]}{|P|} \)
- **Macro Accuracy**: 
  - Defined as the average accuracy over classes of test problems, where each class consists of test examples derived from a base example.
  - Formula: \( \text{Acc}_{\text{macro}}(M; B) = \frac{\sum_{b \in B} 1 [\bigwedge_{p \in P(b)} [M(p) = s(p)]]}{|B|} \)
- **Normalized Accuracy**: 
  - Measures how a method is affected by distractors, considering its accuracy on base problems.
  - Formula: \( \text{norm}(a_M; M) = \frac{a_M}{n_M} \), where \( n_M \) is the base problem accuracy.

## Implementation Notes

- **Prompt Design**: 
  - For few-shot prompting techniques, use one simple exemplar based on either the original problem or the problem with irrelevant context.
  - For 0-COT, directly present the problem followed by "A: Let’s think step by step:".
- **Self-Consistency**: 
  - Implement by sampling multiple solutions and taking the majority vote.
  - Ensure that the correct answer is included in the sampled solutions for high recall.
- **Instructed Prompting**: 
  - Add specific instructions to ignore irrelevant information, which significantly impacts model behavior and performance.
