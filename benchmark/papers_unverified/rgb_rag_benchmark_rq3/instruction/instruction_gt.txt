## Evaluation Details

- **Accuracy**: 
  - Use exact matching to determine if the generated text contains an exact match to the answer.
- **Rejection Rate**: 
  - Measure the rate at which models output the specific content indicating insufficient information.
  - Use both exact matching and ChatGPT evaluation for assessment.
- **Error Detection Rate**: 
  - Measure whether the model outputs the specific content indicating factual errors in the documents.
- **Error Correction Rate**: 
  - Measure whether the model can provide the correct answer after identifying errors.
