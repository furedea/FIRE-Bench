## Experimental Design

### 1. Sensitivity to Changes in Names and Numbers
- **Procedure**: 
  - Use the GSM-Symbolic dataset to generate questions by altering superficial elements such as names and numerical values.
  - Implement 8-shot Chain-of-Thought (CoT) prompting for each model.
  - For each question, create three variations: one with only names changed, one with only numerical values changed, and one with both changed.
  - Generate 50 samples per template, resulting in 5000 total examples for each benchmark.
- **What to Record**:
