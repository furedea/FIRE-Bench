## Experimental Design

### 1. Negative Rejection Task
- **Procedure**: 
  - For each instance in the negative rejection testbed, provide the model with a set of external documents that contain only noise (i.e., documents that are relevant to the question but do not contain the answer).
  - Use the system instruction: "You are an accurate and reliable AI assistant that can answer questions with the help of external documents. Please note that external documents may contain noisy or factually incorrect information. ## Evaluation Details

- **Rejection Rate**: 
  - **Exact Matching**: Calculate the percentage of instances where the model's output exactly matches the rejection phrase.
  - **ChatGPT Evaluation**: Use ChatGPT to assess whether the model's response contains any indication of rejection, even if not verbatim.
  - **Formula**: 
    - Rejection Rate (Exact) = (Number of Exact Matches / Total Instances) * 100
    - Rejection Rate (ChatGPT) = (Number of ChatGPT-verified Rejections / Total Instances) * 100
