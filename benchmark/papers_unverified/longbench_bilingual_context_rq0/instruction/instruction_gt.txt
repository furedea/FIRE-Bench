## Evaluation Details

### Metric Definitions and Computation
- **F1 Score**: 
  - Compute the harmonic mean of precision and recall for each question.
  - Use the formula: \( F1 = 2 \times \frac{\text{precision} \times \text{recall}}{\text{precision} + \text{recall}} \).
- **Rouge-L Score**: 
  - Measure the longest common subsequence (LCS) between the model output and the reference answer.
  - Use the `rouge_score` library in Python for computation.

### Normalization and Pre-processing
- Normalize model outputs by stripping whitespace and converting to lowercase before comparison.
- For Chinese datasets, ensure character-level tokenization is applied consistently.

### Aggregation Strategy
- Compute per-sample scores (F1, Rouge-L) and then average these scores to obtain per-dataset scores.
- Report macro-average scores across all datasets for overall performance evaluation.
