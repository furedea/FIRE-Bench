- **Procedure**: 
  - Use the Compositional Celebrities dataset to evaluate the compositionality gap.
  - For each question, use a 2-shot prompt with examples from the same category.
  - Evaluate both 2-hop questions and their sub-questions separately.
  - Use a category-specific prompt for each question that contains 2 randomly chosen examples removed from the dataset.
- **What to Record**: 
  - Accuracy of 2-hop questions.
  - Accuracy of sub-questions.

### 2. Elicitive Prompting with Self-Ask
- **Procedure**:
