## Experimental Design

### 1. Inducing Emergent Abilities in Shallow Nonlinear Autoencoders on CIFAR100
- **Procedure**:
  - Use the CIFAR100 dataset to train shallow nonlinear autoencoders with a single hidden layer.
  - Define a discontinuous metric, `Reconstructionc`, to evaluate the reconstruction ability of the autoencoders. This metric is defined as the average number of test data with squared reconstruction error below a threshold `c`.
  - Train the autoencoders with varying numbers of bottleneck units to observe changes in reconstruction ability.
  - Evaluate the reconstruction ability using the `Reconstructionc` metric for different values of `c`.
- **What to Record**:
  - Record the mean squared reconstruction error for each model configuration.
  - Record the `Reconstructionc` scores for each model configuration and each value of `c`.
- **Correspondence**:
  - This experiment corresponds to Figure 7 in the paper, which demonstrates the induced emergent reconstruction ability in shallow nonlinear autoencoders.

## Evaluation Details

- **Metric Definition**:
  - The `Reconstructionc` metric is defined as:
    \[
    \text{Reconstruction}_c(\{x_n\}_{n=1}^N) = \frac{1}{N} \sum_{n=1}^N I[||x_n - \hat{x}_n||^2 < c]
    \]
    where \( I(\cdot) \) is the indicator function, \( x_n \) is the original image, and \( \hat{x}_n \) is the reconstructed image.
- **Normalization and Pre-processing**:
  - Normalize the CIFAR100 images to have pixel values between 0 and 1 before feeding them into the autoencoder.
- **Aggregation Strategy**:
  - Compute the `Reconstructionc` score for each model configuration by averaging over all test samples.
  - Compare the scores across different values of `c` to identify the emergence of abilities.

## Implementation Notes

- **Pitfalls and Caveats**:
  - Ensure that the choice of `c` is appropriate to induce the emergent behavior. The threshold should be set such that it highlights the differences in reconstruction ability across model configurations.
- **Special Logic**:
  - Use a discontinuous metric to emphasize the sharpness of the emergent ability. The choice of `c` is crucial in demonstrating the emergence effect.
- **Order of Operations**:
  - First, train the autoencoders with varying bottleneck sizes.
  - Then, evaluate the reconstruction ability using the `Reconstructionc` metric for each configuration.
  - Finally, analyze the results to identify any emergent abilities induced by the metric choice.
