## Evaluation Details

- **Metric Definitions**:
  - **Perplexity**: Calculated as the exponential of the average negative log-likelihood of the test set tokens. Lower perplexity indicates better language modeling performance.
  - **Zero-shot Task Performance**: For tasks like LAMA, check if the correct answer is within the first five predicted words. For math tasks, check if the first number predicted matches the expected result.

- **Normalization and Pre-processing**:
