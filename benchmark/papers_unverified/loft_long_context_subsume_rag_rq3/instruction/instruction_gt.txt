## Experimental Design

### 1. Many-shot ICL Performance Comparison
- **Procedure**: 
  - Use the datasets BBH-date, BBH-salient, BBH-tracking7, BBH-web, and LIB-dialogue.
  - For each dataset, construct a shared many-shot ICL context ensuring that training examples in smaller contexts are included in larger ones.
  - Ensure each class is represented at least once in the context.
  - Use a maximum context length of 32k tokens for BBH datasets and up to 128k or 1M for LIB-dialogue.
  - Present the model with a series of examples (many-shot) before querying it with a new instance.
- **What to Record**: 
  - Record the classification accuracy for each dataset.
  - Capture the model's raw outputs and the predicted class for each query.
- **Correspondence**: 
  - This experiment corresponds to Section 4.6 and Table 2 in the paper.

## Evaluation Details

- **Metric Definitions**: 
  - Use classification accuracy as the primary metric, defined as the number of correct predictions divided by the total number of predictions.
- **Normalization/Pre-processing**: 
  - Ensure that the input examples are tokenized consistently across different context lengths.
  - Normalize the model outputs to ensure they are in a comparable format for accuracy computation.
- **Aggregation Strategy**: 
  - Compute accuracy per dataset and then average across all datasets to get an overall performance measure.

## Implementation Notes

- **Prompt Templates**: 
  - Use specific prompt templates that include task-specific instructions and examples formatted to guide the model's response.
  - Ensure that the prompt includes a clear delimiter between examples and the query.
- **Order of Operations**: 
  - Present the examples in the prompt before the query to simulate many-shot learning.
  - Ensure that the context is constructed such that smaller contexts are subsets of larger ones, maintaining consistency across different context lengths.
- **Pitfalls**: 
  - Be cautious of the model's tendency to rely on parametric knowledge rather than the provided context, especially in tasks with well-known datasets.
