## Experimental Design

### 1. Non-English Masked Language Modeling (MLM) Evaluation
- **Procedure**: 
  - Use the Wiki-40B dataset, which covers 41 languages.
  - Perform whole word masking on 15% of the words in the test set.
  - Evaluate the models in a zero-shot manner without further training.
  - Models to evaluate: BERT, RoBERTa, T5, mBERT, XLM-R.
- **What to Record**: 
  - Bits per character (BPC) for each model across all languages.

### 2. POS Performance Across Languages (Probing and Fine-tuning)
- **Procedure**:

  - For probing, train a linear classifier to predict POS tags from the final layer of each model's encoder.
  - For fine-tuning, unfreeze the encoder weights and allow them to update during training.

## Evaluation Details

### Non-English MLM Evaluation
- **Metric**: 
  - Bits per character (BPC) is used to measure model performance, allowing comparison between models with different tokenizations.
- **Normalization**: 
  - Ensure consistent tokenization across models by using whole word masking.
- **Aggregation Strategy**: 
  - Calculate BPC per language and aggregate results to compare overall performance across models.

### POS Performance Evaluation
- **Metric**: 
  - Accuracy is used to evaluate POS tagging performance.
- **Normalization**: 
  - For words tokenized into multiple subword units, average the representation of all tokens as input to the classifier.
- **Aggregation Strategy**: 
  - Report average accuracy across five runs for each language and model configuration.
