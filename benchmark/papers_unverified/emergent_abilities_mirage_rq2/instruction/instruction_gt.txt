## Experimental Design

### 1. Inducing Emergent Reconstruction Ability in Shallow Nonlinear Autoencoders on CIFAR100
- **Procedure**: 
  - Use shallow nonlinear autoencoders with a single hidden layer.
  - Train these autoencoders on the CIFAR100 dataset.
  - Define a discontinuous metric, `Reconstructionc`, which measures the network's ability to reconstruct a dataset as the average number of test data with squared reconstruction error below a threshold `c`.
  - Vary the number of bottleneck units in the autoencoder to observe changes in reconstruction ability.
- **What to Record**: 
  - Record the mean squared reconstruction error for each model configuration.
  - Record the `Reconstructionc` metric for different values of `c`.
- **Correspondence**: 
  - This experiment corresponds to Figure 7 in the paper, which demonstrates the induced emergent reconstruction ability.

### 2. Inducing Emergent Classification Ability in Autoregressive Transformers on Omniglot
- **Procedure**: 
  - Use autoregressive Transformers to classify Omniglot handwritten characters.
  - Embed Omniglot images using convolutional layers, then feed sequences of embedded image-image class label pairs into decoder-only transformers.
  - Measure image classification performance using subset accuracy: 1 if all images in a sequence of length `L` are classified correctly, 0 otherwise.
  - Vary the sequence length `L` to observe changes in classification ability.
- **What to Record**: 
  - Record the accuracy for each sequence length `L`.
  - Record the subset accuracy for different model configurations.
- **Correspondence**: 
  - This experiment corresponds to Figure 8 in the paper, which shows the induced emergent classification ability.

## Evaluation Details

- **Reconstructionc Metric**: 
  - Defined as `Reconstructionc({xn}Nn=1) = (1/N) * Σ I[||xn - ˆxn||^2 < c]`, where `I(·)` is an indicator function and `ˆxn` is the autoencoder's reconstruction of `xn`.
  - Choose specific thresholds `c` to evaluate the emergent ability.

- **Subset Accuracy**: 
  - Defined as 1 if all `L` images in a sequence are classified correctly, 0 otherwise.
  - This metric is used to evaluate the classification ability of autoregressive Transformers on the Omniglot dataset.

## Implementation Notes

- **Pitfalls**: 
  - Ensure that the choice of metric does not inadvertently introduce biases that could be mistaken for emergent abilities.
  - Carefully select the threshold `c` in the `Reconstructionc` metric to avoid arbitrary results.

- **Order of Operations**: 
  - For both experiments, ensure that the models are evaluated using the defined metrics after training is complete.
  - Adjust the number of bottleneck units or sequence lengths systematically to observe the effects on the emergent abilities.
