## Experimental Design

### 1. Response Bias Evaluation
- **Procedure**: 
  - Generate question pairs (q, q′) from the ATP dataset, where q is the original question and q′ is modified to induce a specific response bias. 
  - For each bias type (acquiescence, allow/forbid asymmetry, response order, opinion floating, odd/even scale effects), create a set of question pairs.
  - Collect 50 responses per question pair from each model using a temperature of 1.
  - Use a prompt template that appends an alphabetical letter to each response option and instructs the model to answer with one of the provided options.
- **What to Record**: 
  - Raw outputs of model responses for each question pair.
  - Count of responses for each option in both q and q′.
  - Calculate the change in response distribution (∆b) for each bias type.

### 2. Non-Bias Perturbation Evaluation
- **Procedure**: 
  - Generate question pairs (q, q′) where q′ includes non-bias perturbations (typos, letter swaps, middle randomization).

- **What to Record**:

### 3. Representativeness Evaluation
- **Procedure**:
