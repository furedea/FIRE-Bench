## Experimental Design

### 1. Zero-shot Task Generalization in Unseen Languages
- **Procedure**: 
  - Evaluate the zero-shot task performance of BLOOMZ and mT0 models on languages that were not intentionally included during pretraining or finetuning.
  - Use the XNLI and XCOPA datasets for evaluation.
  - For each dataset, select languages with minimal representation in the pretraining corpus, such as Thai, Turkish, and Greek.
  - For XNLI, compute the log-likelihood of each possible completion and select the highest scoring option.
  - Use 5 random prompts from PromptSource for each language split and report the median result.
- **What to Record**: 
  - Record the accuracy for each language and task.
  - Collect raw outputs and log-likelihood scores for each completion.
- **Correspondence**: 
  - This experiment corresponds to Figure 5 and Section 4.2 in the paper.

## Evaluation Details

- **Metrics**: 
  - Use accuracy as the primary metric for evaluating zero-shot task performance.
  - For tasks involving multiple-choice questions, use rank classification based on log-likelihood scores.
- **Normalization**: 
  - Normalize log-likelihood scores by the length of the target to ensure fair comparison across different target lengths.
- **Aggregation Strategy**: 
  - Compute per-sample accuracy, then aggregate to per-language accuracy, and finally report the overall accuracy across all evaluated languages.

## Implementation Notes

- **Prompt Selection**: 
  - Use 5 random prompts from PromptSource for each language split to ensure variability and robustness in evaluation.
  - Do not tune prompts based on validation data to maintain the integrity of zero-shot evaluation.
- **Output Parsing**: 
  - Ensure that the model outputs are parsed correctly to extract the predicted labels for accuracy computation.
- **Order of Operations**: 
  - First, evaluate the model on each language using the selected prompts.
  - Then, compute the log-likelihood scores and normalize them.
  - Finally, aggregate the results and report the median accuracy for each language.
