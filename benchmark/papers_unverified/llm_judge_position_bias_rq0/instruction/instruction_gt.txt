## Experimental Design

### 1. Positional Bias Evaluation
- **Procedure**: 
  - For each question in the Vicuna Benchmark, generate two response prompts using the evaluation template: one with Assistant 1's response first and another with Assistant 2's response first.
  - Use GPT-4 and ChatGPT as evaluators to score the responses in both orders.
  - Repeat the process for all 80 questions in the benchmark.
- **What to Record**: 
  - Record the scores assigned to each response in both positions.
  - Calculate the Conflict Rate by comparing the scores for each response pair when their positions are swapped.
- **Correspondence**: This experiment corresponds to Table 2 and Section 2.2 in the paper.

### 2. Multiple Evidence Calibration (MEC)
- **Procedure**: 
  - Use the evidence calibration template to prompt GPT-4 and ChatGPT to generate evaluation evidence before scoring.
  - For each response pair, sample k evaluation scores (k=3) by setting the temperature to 1.
- **What to Record**: 
  - Record all sampled scores and the evaluation evidence.
  - Calculate the average score for each response.
- **Correspondence**: This experiment corresponds to Section 3.1 and Figure 3(a) in the paper.

### 3. Balanced Position Calibration (BPC)
- **Procedure**: 
  - For each response pair, create two prompts by swapping the positions of the responses.
  - Use the MEC strategy to obtain scores for both positions.
  - Calculate the final score as the average of scores from both positions.
- **What to Record**: 
  - Record the scores for each response in both positions.
  - Calculate the final calibrated scores.
- **Correspondence**: This experiment corresponds to Section 3.2 and Figure 3(b) in the paper.

### 4. Human-in-the-Loop Calibration (HITLC)
- **Procedure**: 
  - Calculate the Balanced Position Diversity Entropy (BPDE) for each response pair using the results from MEC and BPC.
  - Select the top-β% examples with the highest BPDE scores for human annotation (β=20%).

- **What to Record**: 
  - Record BPDE scores and the selected examples for human annotation.
  - Record the final evaluation results after incorporating human feedback.
- **Correspondence**: This experiment corresponds to Section 3.3 and Figure 3(c) in the paper.

## Evaluation Details

### Conflict Rate Calculation
- **Formula**: 
  \[
  \text{Conflict Rate} = \frac{\sum_{i=1}^{N} I(\text{ER}_{r1r2}^i \neq \text{ER}_{r2r1}^i)}{N}
  \]
  where \(I(.)\) is the indicator function, and \(N\) is the total number of examples.
- **Aggregation Strategy**: Calculate the Conflict Rate for each model and condition, then average across all examples.

### Balanced Position Diversity Entropy (BPDE)
- **Formula**: 
  \[
  \text{BPDE} = \sum_{\text{er} \in \{\text{win, tie, lose}\}} -p_{\text{er}} \log p_{\text{er}}
  \]
  where \(p_{\text{er}}\) is the proportion of each evaluation result type.
- **Aggregation Strategy**: Calculate BPDE for each example and use it to select examples for human annotation.
