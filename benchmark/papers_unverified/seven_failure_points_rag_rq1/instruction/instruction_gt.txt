## Experimental Design

### 1. Biomedical Question and Answer System
- **Procedure**: 
  - Use the BioASQ dataset, which includes 4017 open access documents and 1000 questions.
  - Index all documents by splitting them into chunks and creating embeddings for each chunk.
  - For each question, convert it into an embedding and retrieve the top-k similar documents using cosine similarity.
  - Pass the retrieved documents to GPT-4 to generate answers.
  - Validate the generated answers using OpenAI Evals.
  - Manually inspect 40 issues and all issues flagged as inaccurate by OpenAI Evals.
- **What to Record**: 
  - Raw outputs from GPT-4.
  - Evaluation results from OpenAI Evals.
  - Manual inspection notes on discrepancies and flagged inaccuracies.
