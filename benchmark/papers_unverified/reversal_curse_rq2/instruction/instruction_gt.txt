## Evaluation Details

- **Exact-Match Accuracy**: 
  - Compute the percentage of test instances where the model's output exactly matches the expected answer.
  - Use temperature zero for generation to ensure deterministic outputs.

- **Log-Probability Analysis**: 
  - Compare the log-probabilities assigned to the correct name versus a random name.
  - Use statistical tests (t-test and Kolmogorov–Smirnov test) to determine if there is a significant difference.
