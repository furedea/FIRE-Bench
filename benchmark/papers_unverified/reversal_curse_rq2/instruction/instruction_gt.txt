## Evaluation Details

- **Exact-Match Accuracy**: 
  - Compute the percentage of test instances where the model's output exactly matches the expected answer.
  - Use temperature zero during evaluation to ensure deterministic outputs.

- **Log-Probability Analysis**: 
  - Compare the log-probabilities assigned to the correct name versus a random name for each prompt.
  - Use statistical tests (t-tests and Kolmogorov–Smirnov tests) to determine if there is a significant difference in log-probabilities.
