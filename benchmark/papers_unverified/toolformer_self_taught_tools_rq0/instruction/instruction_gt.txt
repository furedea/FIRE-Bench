## Evaluation Details

### Metric Definitions and Computation
- **Perplexity**: Compute the weighted cross-entropy loss over tokens. Use the formula `Li(z) = -Σ(wj-i * log pM(xj | z, x1:j-1))` for loss computation.
- **Accuracy**: For tasks like LAMA and question answering, check if the correct answer is within the first few predicted tokens.
- **Lenient Match**: For LAMA, check if the correct word is within the first five predicted words.
- **Aggregation Strategy**: Aggregate results per sample, then per condition (e.g., with/without API calls), and finally overall.
