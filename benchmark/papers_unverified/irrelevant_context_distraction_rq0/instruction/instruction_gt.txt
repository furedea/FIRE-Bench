## Evaluation Details

- **Micro Accuracy**: 
  - Defined as the average accuracy over all test problems.
  - Computed as: \( \text{Acc}_{\text{micro}}(M; P) = \frac{\sum_{p \in P} 1 [M(p) = s(p)]}{|P|} \)
- **Macro Accuracy**: 
  - Defined as the average accuracy over classes of test problems, where each class consists of test examples derived from a base example.
  - Computed as: \( \text{Acc}_{\text{macro}}(M; B) = \frac{\sum_{b \in B} 1 [\bigwedge_{p \in P(b)} [M(p) = s(p)]]}{|B|} \)
- **Normalized Accuracy**: 
  - Measures how a method is affected by distractors, considering its accuracy on base problems.
  - Computed as: \( \text{norm}(a_M; M) = \frac{a_M}{n_M} \), where \( n_M \) is the base problem accuracy of method M.

## Implementation Notes

- **Prompt Design**: 
  - For few-shot prompting techniques, use one simple exemplar based on either the original problem or the problem with irrelevant context.
  - For 0-COT, directly present the problem followed by "A: Let’s think step by step:".
- **Self-Consistency**: 
  - Implemented by sampling multiple solutions and taking the majority vote.
  - Use 20 samples per problem with a temperature of 0.7 for sampling.
- **Instructed Prompting**: 
  - Add specific instructions to the prompt to guide the model in ignoring irrelevant context.
  - Ensure that the instruction is clear and directly related to the task at hand.
