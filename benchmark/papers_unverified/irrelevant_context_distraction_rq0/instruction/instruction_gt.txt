## Evaluation Details

- **Micro Accuracy**: 
  - Defined as the average accuracy over all test problems.
  - Formula: \( \text{Acc}_{\text{micro}}(M; P) = \frac{\sum_{p \in P} 1 [M(p) = s(p)]}{|P|} \)
  - Each problem is weighted equally.

- **Macro Accuracy**: 
  - Defined as the average accuracy over classes of test problems, where each class consists of test examples derived from a base example.
  - Formula: \( \text{Acc}_{\text{macro}}(M; B) = \frac{\sum_{b \in B} 1 [\bigwedge_{p \in P(b)} [M(p) = s(p)]]}{|B|} \)
  - A class is considered correct if all problems in the class are solved correctly.

- **Normalized Accuracy**: 
  - Measures the effect of distractors relative to the base problem accuracy.
  - Formula: \( \text{norm}(a_M; M) = \frac{a_M}{n_M} \)
  - \( n_M \) is the base problem accuracy of method M.

## Implementation Notes

- **Prompt Design**: 
  - For COT, LTM, and PROGRAM, use a single exemplar problem in the prompt.
  - For 0-COT, directly present the problem followed by "A: Let’s think step by step:".
  - Include irrelevant context in the exemplar for some experiments to test robustness.
  - Use the instruction "Feel free to ignore irrelevant information given in the questions." to guide the model in some experiments.

- **Order of Operations**: 
  - Ensure that the irrelevant context is added consistently across all problems.
  - Verify that the added context does not change the standard solution of the base problem.
  - For self-consistency, ensure that the majority vote is taken from the sampled solutions. If there is a tie, select a random top-tier result.
